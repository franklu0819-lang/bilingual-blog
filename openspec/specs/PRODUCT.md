# Product Specification: Bilingual Technical Blog

## Overview
A sleek, modern technical blog that supports both Chinese and English languages, optimized for developers and technical writers. The blog should be easy to maintain and deployable to Vercel.

## Core Features
- **Bilingual Support**: Toggle between Chinese (Simplified) and English.
- **Blog Posts**: List of blog posts with metadata (date, tags, description).
- **Post Reading**: Full MDX/Markdown support for posts.
- **Responsive Design**: Mobile-first, looks great on all devices.
- **SEO Optimized**: Metadata for both languages.
- **Fast Loading**: Built with Next.js for static/server-side rendering.

## Pages
- **Home**: Latest posts list.
- **Blog**: Filterable list of posts.
- **About**: Introduction to the author.
- **Projects**: Portfolio or project showcase.
