# Tech Stack Specification

## Frontend
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Content Management
- **Format**: MDX (Markdown with React components)
- **Library**: Contentlayer (or next-mdx-remote if Contentlayer is too heavy)

## Internationalization (i18n)
- **Library**: next-intl or a custom lightweight i18n solution.

## Deployment
- **Platform**: Vercel
